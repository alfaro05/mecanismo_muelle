El proyecto se desarrolló mediante Django. En el interior del directorio "interfaz" se encuentran los archivos de la aplicación web.

El archivo "mechanisms.py" se encuentra el modelo del mecanismo y los eslabones, con sus respectivas clases y métodos.

La administración y manipulación de la información suministrada por el usuario sigue el modelo MVT de Django. La lógica de este funcionamiento se encuentra principalmente en el archivo "views.py", en el directorio "interfaz".
